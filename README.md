## Getting Started

### Prerequisites

- Ruby
- Bundler
- Jekyll

### Installation

#### Ruby

Ruby 공식 홈페이지를 참고하여 각 OS에 맞게 Ruby를 설치한다.

#### Bundler && Packages

```bash
gem install bundler

bundle install
```

#### Packages

```bash
npm i
```

### Usage

```bash
bundle exec jekyll serve
```
